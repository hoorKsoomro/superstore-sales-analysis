# 📊 Superstore Sales Analysis

## 📌 Project Overview

The objective of this project was to analyze Superstore sales data using SQL and Pandas to uncover meaningful business insights. The analysis focused on identifying top-performing states, cities, customer segments, products, sales trends over time, and shipping performance. Based on these findings, business recommendations were provided to support data-driven decision-making.

# 📈 Business Questions Answered

- Which states generate the highest sales?
- Which cities contribute the most revenue?
- Which product categories perform the best?
- Which customer segments generate the highest sales?
- Which customers contribute the most revenue?
- Is the business dependent on a few customers?
- Is the business dependent on a few products?
- How do sales change over months, quarters, and years?
- Which shipping method is the fastest?
