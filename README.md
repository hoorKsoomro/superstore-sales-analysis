# 📊 Superstore Sales Analysis

## 📌 Project Overview

The objective of this project was to analyze Superstore sales data using SQL and Pandas to uncover meaningful business insights. The analysis focused on identifying top-performing states, cities, customer segments, products, sales trends over time, and shipping performance. Based on these findings, business recommendations were provided to support data-driven decision-making.
